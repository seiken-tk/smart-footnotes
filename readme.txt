WordPressの投稿に脚注を簡単に追加できるプラグインです。

== Description ==

SmartFootnotesは、WordPressの投稿やページに脚注を簡単に追加できるプラグインです。
ショートコードを使用して脚注を追加し、自動的に番号付けされた脚注を記事の末尾に表示します。

主な機能：

* シンプルなショートコードで脚注を追加
* 脚注の自動番号付け
* ホバーで脚注内容をポップアップ表示
* クリックで脚注へジャンプ（戻るリンク付き）
* 管理画面でスタイルのカスタマイズが可能
* マルチサイト対応
* 多言語対応

== Installation ==

1. プラグインを「wp-content/plugins/」ディレクトリにアップロードするか、WordPress管理画面からプラグインを直接インストールします。
2. WordPress管理画面でプラグインを有効化します。
3. 設定 > SmartFootnotes から、必要に応じて設定を行います。

== Frequently Asked Questions ==

= 脚注の追加方法は？ =

[sfnote]脚注の内容[/sfnote]というショートコードを使用します。
例：この文章に脚注を追加します[sfnote]これが脚注の内容です[/sfnote]。

= 脚注のスタイルをカスタマイズできますか？ =

はい、管理画面の設定から以下の項目をカスタマイズできます：
* 脚注の表示スタイル
* ポップアップの表示スタイル
* 戻るリンクの文字
* カスタムCSS

= 脚注一覧の表示位置を変更できますか？ =

デフォルトでは記事の末尾に表示されます。表示位置の変更は現在のバージョンではサポートしていません。
